Další metody pro zobrazení dat

Data můžeme v JavaScriptu zobrazovat několika způsoby. Jeden už umíme a to .innerHTML.

způsoby zobrazení dat:

  • innerHTML - vypíše data do nějakého HTML elementu
  • document.write() - vypíše data do HTML výstupu
  • window.alert() - vypíše data do alert boxu
  • console.log() - vypíše data do konzole prohlížeče

Metoda document.write()

Tato metoda je vhodná pro testování. Pokud použijeme metodu document.write(), až po tom co se načte HTML dokument, tak se celý HTML kód vymaže a zobrazí se pouze výstup metody document.write().
