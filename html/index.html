JavaScript- úvod

Co je JavaScript?

  • JavaScript je programovací jazyk používaný při tvorbě webových stránek (pozor! JavaScript není Java má pouze podobnou syntaxi.)

Proč se učit JavaScript?

  • je jednoduchý na naučení
  • je zábavný - JavaScript je hlavně používán na vývoj frontendu webu a je zde spoustu nástrojů, díky kterým můžete rychle vytvořit fungující prototyp
  • je to jeden z nejvíce používaných programovacích jazyků a je všude s JavaScriptem můžete designovat vizuální efekty stránky

Co můžu dělat s JavaScriptem?

  • s JavaScriptem můžeš přidat webové stránce interaktivitu, dnes téměř každá moderní webová stránka má v sobě JavaScript
  • s JavaScriptem můžeš vytvářet webové aplikace, prohlížečové hry, ...
