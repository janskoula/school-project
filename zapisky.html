metoda getElementById()

Jednou ze spousty JavaScript HTML metod je getElementById().
Příklad: document.getElementById("pokus").innerHTML = "Hello World!"

- abychom mohli mít přístup k HTML elementu použijeme metodu "document.getElementById(id)"
- atribut id definuje HTML element s kterým se bude pracovat
- vlastnost: "innerHTML" definuje obsah, který se vypíše do elementu

<p id=”pokus”></p> - jak vidíte tento tag v sobě nená žádný text, ale když spustíte stránku JavaScript do tohoto tagu vypíše "Hello World!"

Aby JavaScript mohl vůbec něco na vaší stránce něco dělat musíte ho připojit k HTML kódu. JavaScript můžete připojit dvěmi možnostmi. Buď JavaScript napíšete přímo v HTML kódu mezi tegy "<script> </script>" nebo ho můžete připojit v externím souboru.

celý kód s JavaScriptem v HTML kódu: 

<!DOCTYPE html>
<html>
	<head>
		<title>JavaScript přímo v HTML kódu</title>
	</head>
	<body>
		<p id=”pokus”></p>

		<script>
			document.getElementById(“pokus”).innerHTML = “Hello World”;
		</script>
	</body>
</html>

celý kód s externím JavaScript kódem: 

<!DOCTYPE html>
<html>
	<head>
		<title>JavaScript v externím souboru</title>
	</head>
	<body>
		<p id=”pokus”></p>

		<script src=”externiJavaScript.js”></script>
	</body>
</html>

- v tomto případě musíme mít ve složce kde máte uložený HTML kód i soubor s JavaScript kódem
- pokud máte JavaScript kód někde jinde musíte k němu specifikovat cestu - například: "/js_kody/externiJavaScript.js" - tento JavaScript kód se nachází ve složce js_kody, která se nachází ve složce s HTML kódem
- dále tento soubor musí mít koncovku .js což je označení JavaScript souboru

kód v souboru “externiJavaScript.js”: 

document.getElementById(“pokus”).innerHTML = “Hello World”;

výhodou přidáváním JavaScriptu externě je:

- jednodušší znovupoužití na jiné stránce
- HTML kód je přehlednější
